Q1: Why might we consider using Git?

Answer: Is a version control system where you can manage code/data changes. 
    It is also useful when working with a team. Can also reverse to an earlier code version.


Q2: What field could data science be applied to? Give an Example.

Answer: Healthcare. For example, analyzing patient data and predicting risk of diseases for certain patients, etc.


Q3: What is the difference between Merging and Rebasing?

Answer: For Merging, the changes from the source branch is combined into the target branch. 
    For Rebasing, the changes are actually moved from the source branch to the target branch.
    Merging also preserves the history, while Rebasing overwrites it.
